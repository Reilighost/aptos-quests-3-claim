# OAT Claimer for Aptos-Campaign-Quest-3

This README provides instructions for setting up and utilizing the OAT claimer for Aptos campaign's Quest3. The guide below will assist you in initiating the process.

### Installation

1. **Install Dependencies**

   Begin by cloning the repository to your local environment and navigating into the directory. Install the necessary Python packages by executing the installation command ```pip install -r requirements.txt```

2. **Configure Private Keys**

   Insert your private key in a file `keys_apt.txt`. Post-claim, the keys are moved to a separate file.

3. **Smartproxy Setup**

   Register at [Smartproxy](https://dashboard.smartproxy.com). Following purchase, incorporate youre IP into the whitelist at this [page](https://dashboard.smartproxy.com/mobile-proxies/proxy-setup).

4. **2Captcha Setup**

   Visit [2Captcha](https://2captcha.com), top-up your account balance, obtain you API key and insert into `config.py`.

### Usage

Upon completing the setup, activate the claimer by executing the main script.
